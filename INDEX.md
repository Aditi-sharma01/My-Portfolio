## 🔍 Troubleshooting Guide

### Common Issues

**"npm install fails"**
- See [QUICK_START.md](QUICK_START.md#phase-1-installation--setup-15-mins)
- Check Node.js version requirements in [README.md](README.md#prerequisites)

**"Styles look wrong"**
- Check Tailwind config in [PROJECT_STRUCTURE.md](PROJECT_STRUCTURE.md)
- Review global styles in `src/index.css`

**"How do I change X?"**
- Refer to [CUSTOMIZATION.md](CUSTOMIZATION.md) quick reference table
- Find code examples in that section

**"Website won't load on deployment"**
- Check [DEPLOYMENT.md](DEPLOYMENT.md#troubleshooting) troubleshooting section
- Verify build commands and settings

**"Form doesn't work"**
- See [DEPLOYMENT.md](DEPLOYMENT.md#form-backend-setup) for options
- Choose and configure your backend
